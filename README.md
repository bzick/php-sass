SASS PHP Extension
========

Installation:

    mkdir /tmp/php-sass
    git clone https://github.com/bzick/php-sass.git /tmp/php-sass
    cd /tmp/php-sass
    sudo ./install.sh
    # or 
    sudo ./checkinstall.sh
    
See install.sh or checkinstall.sh for more information
